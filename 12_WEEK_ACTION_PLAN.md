## ADAPTATION GUIDE

### If You're Ahead of Schedule:

**Earning more than targets:**
- Don't burn out! Maintain pace.
- Raise rates faster
- Be more selective with clients
- Start delegation earlier

### If You're Behind Schedule:

**Earning less than targets:**

**Week 1-2 behind**: Normal, keep pushing
**Week 3-4 behind**: Diagnose the problem
- Not enough applications? Apply to 30/day
- Not getting hired? Fix proposals (use templates)
- Not enough time? Cut non-essentials

**Month 1 behind**: Adjust expectations
- Maybe start 20 hours/week, not 40
- Maybe focus on 2 platforms, not 5
- Extend timeline: 16 weeks instead of 12

**Key**: Progress > Perfection

---

## EMERGENCY PROTOCOLS

### If You're Burning Out:

**Immediate Actions**:
1. Take full day off (today)
2. Sleep 10 hours
3. Get outside
4. Talk to someone

**This Week**:
1. Reduce hours 30%
2. Drop one platform
3. Add one fun activity daily
4. Re-read MINDSET_MOTIVATION_GUIDE.md

### If You're Not Making Any Money:

**Week 1-2**: Keep applying, this is normal
**Week 3**: Diagnostic
- Are you approved on platforms? If no, apply to easier ones
- Are you applying enough? If no, 30/day minimum
- Are proposals good? If no, use templates

**Week 4**: Pivot
- If freelance isn't working: Focus on delivery/services
- If online isn't working: Try offline (TaskRabbit, etc.)
- If complex skills: Simplify (data entry, surveys)

### If You Get Sick/Emergency:

**Short-term (1-3 days)**:
- Communicate with clients ASAP
- Ask for extensions
- Pause new work
- Rest and recover

**Long-term (1+ weeks)**:
- Let clients know timeline
- Refund if you can't deliver
- Pause all platforms
- Health > Money
